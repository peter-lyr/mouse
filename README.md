# 240521-00h56m

## 概要

除了热键外,其他所有鼠标操作,都要经过按下鼠标右键

按下鼠标右键后,可以通过在不同的区域内,以以下三种方式,去做不同的事情

- `松开右键`
- `向上滚动滚轮`
- `向下滚动滚轮`

## 失效

按下鼠标右键后,以下事件会导致其他事件失效

- `滚动滚轮`,会让`松开右键`的事件失效
- `按下左键`或者`中键`,会让以上三种事件均失效
